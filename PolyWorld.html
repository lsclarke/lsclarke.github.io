Poly-World

Poly-World is a 3D action adventure game that was created using the Unreal 4 Engine In my Senior class. In a 8 week time period I created and interactive player environment that would encourage the player to use the Parkour mechanics I made to help find the special cubs to beat the game.

Time frame: 8 weeks
Solo Project
Game Engine: Unreal Engine
Programming Language: Blueprint Scripting
